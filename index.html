¿Qué es la violencia psicológica?

La violencia psicológica es cualquier acción o conducta que afecta la estabilidad emocional de una persona mediante insultos, amenazas, humillaciones, manipulación, intimidación o control excesivo.

Aunque no deja marcas físicas visibles, puede generar consecuencias profundas como ansiedad, inseguridad, baja autoestima y dificultades para relacionarse con otras personas.

Tipos de violencia

Violencia Psicológica

Humillaciones, manipulación, amenazas e insultos.

Violencia Física

Daño corporal causado por golpes o agresiones.

Violencia Económica

Control del dinero para limitar la independencia.

Violencia Digital

Acoso o amenazas mediante internet y redes sociales.

Consecuencias de la violencia psicológica

  • Baja autoestima.
  • Ansiedad.
  • Estrés emocional.
  • Depresión.
  • Aislamiento social.
  • Problemas escolares.
  • Dificultades para relacionarse.

Prevención

La prevención comienza identificando señales de manipulación, humillación o amenazas. Es importante pedir ayuda a familiares, maestros o instituciones especializadas.

Promover relaciones basadas en el respeto y la empatía contribuye a crear ambientes seguros para todos.

Nuestro objetivo

Informar y concientizar sobre la violencia psicológica para fomentar el respeto, la empatía y la prevención. Conocer este problema es el primer paso para construir una sociedad más segura y libre de violencia.
